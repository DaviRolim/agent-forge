Build a todo app with add, complete, delete, and filter functionality (all/active/completed). Todos must persist in localStorage across page refreshes. Include a "clear completed" button and a counter showing remaining active items. Double-click a todo to edit its text inline.

## Design Constraints
- Stack: Next.js + Tailwind CSS
- Mobile-first, responsive
- Inspired by TodoMVC but with a modern, polished look
- Smooth animations on add/remove (fade or slide)

Build a sales dashboard with:

- **Header stat cards:** Total Revenue, Total Orders, Total Customers, Average Order Value — each with an icon and percentage change indicator (up/down arrow + green/red)
- **Line chart:** Monthly revenue for the past 12 months
- **Bar chart:** Top 5 products by sales volume
- **Recent orders table:** 10 rows with columns: Order ID, Customer Name, Product, Amount, Date, Status (badge). Sortable by Date and Amount columns.

Use recharts for the charts. All data should be realistic mock data (not lorem ipsum). Include a dark/light mode toggle.

## Design Constraints
- Stack: Next.js + Tailwind CSS
- Mobile-first, responsive (charts stack vertically on mobile)
- Clean dashboard aesthetic — think Linear or Vercel dashboard
- Cards should have subtle hover effects
